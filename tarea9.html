Simbolos

Los simbolos mas usados en PHP:

        
Operadores aritméticos: + (suma), - (resta), * (multiplicación), / (división), % (módulo), ** (exponenciación).

Operadores de asignación: = (asignación), += (suma y asignación), -= (resta y asignación), *= (multiplicación y asignación), /= (división y asignación), etc.

% Para el modulo.

Operadores de comparación: == (igual), === (idéntico), != o <> (no igual), !== (no idéntico), < (menor que), > (mayor que), etc.

Operadores lógicos: && o and (y lógico), || o or (o lógico), ! (no lógico).

Otros operadores: .= (concatenación y asignación), ., etc.

Sintaxis

        
Un script de PHP comienza con <?php y termina con ?>.
El código PHP se coloca entre estas etiquetas.
Las declaraciones en PHP terminan con un punto y coma (;).
Los comentarios en PHP comienzan con // para una sola línea, o se encierran entre /* y */ para múltiples líneas.
PHP no distingue entre mayúsculas y minúsculas para sus palabras clave, pero las variables sí son sensibles a las mayúsculas y minúsculas.

Esqueleto

        
<?php
// Aquí va el código PHP
?>

Paradigma

PHP es un lenguaje de programación multiparadigma. Soporta programación imperativa, funcional, orientada a objetos, procedimental y reflexiva.

Aplicaciones para ejecutar código

Puedes usar cualquier editor de texto para escribir código PHP, como Sublime Text, Atom, Visual Studio Code, etc.

Para compilar y ejecutar código PHP, necesitas un servidor web como Apache o Nginx, y también necesitas instalar el intérprete de PHP en tu servidor.

Ejemplo de código para calcular el factorial de un número

      
<?php
function factorial($n) {
  if ($n == 0) {
    return 1;
  } else {
    return $n * factorial($n - 1);
  }
}

echo "El factorial de 5 es: " . factorial(5);
?>
