Simbolos

Algunos de los símbolos más utilizados en R incluyen:

        
+, -, *, /, ^: Operadores aritméticos.
==, !=, <, >, <=, >=: Operadores relacionales.
!, &, &&, |, ||: Operadores lógicos.
<- o =: Operadores de asignación.
%in%: Identifica si un elemento pertenece a un vector.
$: Permite acceder a objetos almacenados dentro de un objeto.

Sintaxis

La sintaxis de R es similar a la de otros lenguajes de programación. Incluye el uso de variables, comentarios y palabras clave2. Los bloques de código se delimitan con llaves {} y las declaraciones se separan con saltos de línea o, opcionalmente, con punto y coma3.

Esqueleto

Un programa en R generalmente consta de la definición de funciones y su llamada para realizar operaciones. Aquí tienes un ejemplo básico: 

      
# Definición de la función
mi_funcion <- function(a, b) {
  resultado <- a + b
  return(resultado)
}

# Llamada a la función
suma <- mi_funcion(3, 4)
print(suma)

Paradigma

R es un lenguaje multiparadigma que soporta programación procedimental, orientada a objetos, funcional, reflexiva e imperativa73.

Puedes usar varios editores y compiladores para R, como RStudio, que es un entorno de desarrollo integrado (IDE) para R. También puedes usar compiladores en línea como los proporcionados por sitios web como Programiz4, W3Schools5 y Replit6.

Ejemplo de código para calcular el factorial de un número

      
# Definición de la función factorial
factorial <- function(n) {
  if(n == 0) {
    return(1)
  } else {
    return(n * factorial(n-1))
  }
}

# Uso de la función factorial
print(factorial(5))
