Simbolos

Haskell utiliza una variedad de símbolos en su sintaxis. Algunos de los más comunes incluyen -> para las firmas de tipo, = para la definición de funciones, :: para especificar tipos y ++ para la concatenación de listas. Los operadores como +, -, *, / se usan para las operaciones matemáticas.

Sintaxis

La sintaxis de Haskell es bastante única en comparación con muchos otros lenguajes de programación debido a su naturaleza funcional. Las funciones se definen con un nombre, seguido de parámetros separados por espacios, un signo igual y luego el cuerpo de la función.

Esqueleto

Un programa Haskell típico comienza con la definición del módulo y las declaraciones de importación, seguidas por las definiciones de funciones y tipos. 

      
module Main where

main :: IO ()
main = putStrLn "¡Hola, mundo!"

Paradigma

Haskell es un lenguaje de programación puramente funcional5. Esto significa que se centra en la evaluación de funciones matemáticas y evita cambiar el estado y los datos mutables.

Aplicaciones para ejecutar código

El compilador más utilizado para Haskell es el Compilador Glasgow Haskell (GHC)4. Para editar código Haskell, puedes usar cualquier editor de texto que prefieras, como Sublime Text, Atom o Visual Studio Code. Hay IDEs como IntelliJ IDEA (con el plugin Haskell) y plugins para otros editores que proporcionan características útiles como resaltado de sintaxis y autocompletado.

Ejemplo de código para calcular el factorial de un número

      
factorial :: Integer -> Integer
factorial 0 = 1
factorial n = n * factorial (n - 1)
