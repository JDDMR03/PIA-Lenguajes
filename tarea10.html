Simbolos

Los simbolos mas usados en SQL son:

        
+ : Suma
- : Resta
* : Multiplicación
/ : División
% : Módulo
= : Igual a
> : Mayor que
< : Menor que
>= : Mayor o igual que
<= : Menor o igual que
<> : No igual a

Sintaxis

SQL es un lenguaje declarativo que se utiliza para comunicarse con una base de datos. Las consultas en SQL comienzan con un verbo que describe la acción, como SELECT, INSERT, UPDATE o DELETE. A continuación , se especifican el sujeto y el predicado. Un predicado especifica condiciones que pueden evaluarse como verdaderas, falsas o desconocidas.

Esqueleto

Un programa SQL típicamente consiste en una serie de declaraciones SQL que realizan tareas como la creación de tablas, la inserción de datos, la actualización de datos y la consulta de datos.

Paradigma

SQL pertenece al paradigma declarativo. Es un lenguaje específico del dominio utilizado en la programación y diseñado para gestionar datos en un sistema de gestión de bases de datos relacionales (RDBMS).
