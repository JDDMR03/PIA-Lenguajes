Simbolos

Los simbolos mas usados en python son:

        
/ Para la división.
** Para la potencia.
% Para el modulo.

Sintaxis

La sintaxis de Python es clara y fácil de leer. Utiliza indentación para delimitar bloques de código en lugar de llaves {}. Las declaraciones terminan con una nueva línea en lugar de un punto y coma ;.

Esqueleto

Un programa Python típico puede comenzar con la importación de módulos usando el comando import. Luego define y llama a las funciones.

Paradigma

Python es un lenguaje de programación multiparadigma. Soporta programación orientada a objetos, programación imperativa y, en menor medida, programación funcional.

Aplicaciones para ejecutar código Python

Puedes ejecutar Python en varias aplicaciones. Algunas opciones populares incluyen IDLE (el entorno de desarrollo integrado de Python), Jupyter Notebook, PyCharm, Atom y Visual Studio Code.

Ejemplo de código para calcular el factorial de un número

      
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

num = 5
print("El factorial de", num, "es", factorial(num))
