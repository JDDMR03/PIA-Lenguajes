Clasificación de los Lenguajes Informáticos

Lenguaje de programación: Permiten crear acciones, ejecutar órdenes y algoritmos para controlar de forma física y lógica un sistema. Ejemplo: Python.

Lenguaje de especificación: Se utilizan para describir el comportamiento de un sistema, sin necesidad de definir cómo se implementa ese comportamiento. Ejemplo: Lenguaje Unificado de Modelado (UML).

Lenguaje de consulta: Se utilizan para consultar datos en una base de datos o sistemas de información. Ejemplo: SQL (Structured Query Language).

Lenguaje de marcas: Permiten poner etiquetas a un documento para darle formato y estructura. Ejemplo: HTML (HyperText Markup Language).

Lenguaje de transformación: Se utilizan para transformar datos de un formato a otro. Ejemplo: XSLT (Extensible Stylesheet Language Transformations).

Protocolo de comunicaciones: Conjuntos de reglas que permiten la comunicación entre dispositivos. Ejemplo: HTTP (Hypertext Transfer Protocol).
